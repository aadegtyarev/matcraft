# Контракт CLI matcraft

Пользовательская поверхность бинарника `matcraft` (v0.6.0). Что гарантируется тому,
кто вызывает инструмент; изменение семантики ниже — изменение контракта (semver).
Данные (инвентарь, домены, аттестация) — `docs/decisions/plutser-sarno-taxonomy.md`.

## Глобальный флаг

| Флаг | Значения | По умолчанию | Семантика |
|---|---|---|---|
| `--mode` | `classic` \| `full` | `classic` | Набор корней для `list-roots`, `generate` (без `--root`), `random`, `root-of-the-day`. Ограничен clap `value_enum`; иное значение → ошибка парсинга (exit 2). |

`classic` = ядро (7) + два продуктивных экскреторных корня (сра-, сса-) = 9 корней.
`full` = все 35 корней. Состав classic выводится из таксономии
(`Nuclear` ∪ {`Excretory` ≤ B}), а не хардкодится.

## Субкоманды

### `explore <корень> [--suffix <s>]`
Полная комбинаторная парадигма корня (18 префиксов × суффиксы × окончания).

- Работает для **любого** из 35 корней независимо от `--mode`.
- Завершающий дефис в корне отбрасывается (`еб-` = `еб`).
- Заголовок: корень, глосс, домен, класс продуктивности.
- Таблица сгруппирована по суффиксу; показывает инфинитивы с аттестацией и значением.
- Чисто именной корень (нет глагольных форм): версионно-нейтральное сообщение об
  отсутствии глагольной парадигмы + лингвистическая заметка.
- `--suffix` фильтрует по значению или отображению суффикса; невалидное значение
  просто не даёт совпадений (не ошибка).
- Неизвестный корень → stderr с сообщением и списком доступных корней, **exit 1**.

### `generate [--root <корень>] [--count <N>]`
Случайные словоформы из пула (аттестация ≠ `Impossible`).

- Без `--root`: пул из корней текущего `--mode`.
- `--count` по умолчанию 1; **> 100 → exit 1**; значение внутренне зажато в 1..=100.
- Неизвестный `--root` → **exit 1**.

### `list-roots`
Список корней текущего `--mode`.

- `full`: сгруппирован по домену (Ядро / Экскреторная / Периферия), с классом
  продуктивности `[A]`–`[E]`; чисто именные помечены «(только именной)».
- `classic`: плоский список из 9 корней.

### `random`
Один случайный корень текущего `--mode` в рамке: глосс, суффиксальные классы, до 3
примеров (инфинитивы уровня common), лингвистическая заметка. Чисто именной корень —
без примеров, с пометкой.

### `root-of-the-day`
Один корень текущего `--mode` в той же рамке, что `random` (глосс, суффиксальные
классы, до 3 примеров-инфинитивов уровня common, лингвистическая заметка).

- **Детерминирован в пределах календарных суток UTC**: любой повторный вызов в те же
  сутки даёт тот же корень; со сменой суток корень меняется.
- Сутки берутся по **UTC** (секунды системных часов с эпохи / 86400), а не по
  локальному часовому поясу — вблизи локальной полуночи смена корня может быть смещена
  относительно локального дня.
- Конкретное соответствие «день → корень» **не** является гарантией стабильности между
  версиями: сид детерминирован в пределах major-версии `rand`, будущий major может
  перетасовать соответствие. Контракт обещает «один корень в пределах суток», а не
  «день X → корень Y навсегда».
- Аргументов нет; чисто именной корень — без примеров, с пометкой (как `random`).

## Уровни аттестации

`common` · `rare` · `possible` · `impossible` — заземлены на синтез Плуцера-Сарно;
`possible` — честная экстраполяция по аналогии (методология в справочном доке).

## Коды возврата

| Код | Значение |
|---|---|
| 0 | Успех |
| 1 | Ошибка приложения (неизвестный корень, count > 100) |
| 2 | Ошибка разбора аргументов (clap) |

## Стабильность

Инструмент офлайн, без сети, файлов и пользовательских данных. Вывод —
детерминирован, кроме `generate`/`random` (случайный выбор каждый вызов).
`root-of-the-day` — промежуточный случай: детерминирован **в пределах суток**, меняется
со сменой суток UTC. Инвентарь из 35 корней и состав `classic` (9) — стабильный
контракт в пределах minor-версий.
