# Модель угроз matcraft

## Методология

Оценка угроз для open-source CLI-инструмента без сетевого взаимодействия,
работы с файлами или пользовательскими данными.

## Таблица угроз

| Поверхность атаки | Описание | Риск | Митигация |
|---|---|---|---|
| Subcommand `explore <ROOT>` | Пользователь передаёт строку корня. Валидируется по списку известных корней | Низкий | Неизвестный корень → exit(1) с сообщением об ошибке и списком доступных корней |
| Subcommand `explore --suffix` | Пользователь передаёт строку суффикса | Низкий | Фильтр по суффиксу; невалидное значение просто не находит совпадений |
| Subcommand `generate --root` | Пользователь передаёт строку корня | Низкий | Валидация по списку известных корней |
| Subcommand `generate --count` | Пользователь передаёт usize | Низкий | Ограничение на 100 (cap в коде). Значение парсится clap |
| Subcommand `list-roots` | Не принимает аргументов | Отсутствует | Н/П |
| Subcommand `random` | Не принимает аргументов | Отсутствует | Н/П |
| Subcommand `root-of-the-day` | Не принимает аргументов | Отсутствует | Н/П |
| Флаг `--mode` | Пользователь передаёт режим (classic/full) | Отсутствует | Ограничен clap `value_enum` двумя значениями; иное значение → ошибка парсинга |
| Секреты | Нет API-ключей, токенов, паролей | Отсутствует | Нет |
| Границы доверия | Нет. Весь ввод контролируется пользователем | Отсутствует | Н/П |
| Инъекции | Нет shell-вызовов, SQL, шаблонизаторов, десериализации | Отсутствует | Н/П |
| Утечка данных | Приложение не собирает, не хранит и не передаёт данные | Отсутствует | Н/П |
| Доступ и авторизация | Локальный CLI-инструмент, без контроля доступа | Н/П | Н/П |
| Зависимости | clap 4.6, rand 0.9 — проверенные, широко используемые библиотеки | Низкий | Пинговка версий; обновление по необходимости |

## Особые риски

### Оскорбительный вывод

Выходные данные инструмента — обсценная лексика русского языка. Это является
**функцией**, а не уязвимостью. Однако:

- Пользователь явно запрашивает генерацию (`matcraft explore <корень>`).
- README предупреждает о характере контента.
- Инструмент не предназначен для несовершеннолетних.

Данный риск не требует митигации, так как является осознанной особенностью
продукта.

## Вывод

Атакующая поверхность минимальна: пять subcommand с 2-3 аргументами и флаг
`--mode`, все валидируются. Проект не обрабатывает конфиденциальные данные, не выполняет
сетевых запросов и не имеет injection-векторов.
Единственный риск — оскорбительный характер вывода — является ожидаемой
особенностью продукта.
