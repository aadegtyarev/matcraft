# Модель угроз matcraft

## Методология

Оценка угроз для open-source CLI-инструмента без сетевого взаимодействия,
работы с файлами или пользовательскими данными.

## Таблица угроз

| Поверхность атаки | Описание | Риск | Митигация |
|---|---|---|---|
| Аргумент `--topic` | Пользователь передаёт строку. Валидируется статическим списком. | Нет риска | Валидация через match по известным темам; неизвестная тема → exit(1) с сообщением об ошибке |
| Секреты | Нет API-ключей, токенов, паролей. | Отсутствует | Нет |
| Границы доверия | Нет. Весь ввод контролируется пользователем. | Отсутствует | Н/П |
| Инъекции | Нет shell-вызовов, SQL, шаблонизаторов, десериализации. | Отсутствует | Н/П |
| Утечка данных | Приложение не собирает, не хранит и не передаёт данные. | Отсутствует | Н/П |
| Доступ и авторизация | Локальный CLI-инструмент, без контроля доступа. | Н/П | Н/П |
| Зависимости | clap 4.6, rand 0.9 — проверенные, широко используемые библиотеки. | Низкий | Пинговка версий; обновление по необходимости |

## Особые риски

### Оскорбительный вывод

Выходные данные инструмента — обсценная лексика русского языка. Это является
**функцией**, а не уязвимостью. Однако:

- Пользователь явно запрашивает генерацию (`--topic <ТЕМА>`).
- README предупреждает о характере генерируемого контента.
- Инструмент не предназначен для несовершеннолетних.

Данный риск не требует митигации, так как является осознанной особенностью
продукта.

## Вывод

Атакующая поверхность отсутствует. Проект не обрабатывает конфиденциальные
данные, не выполняет сетевых запросов и не имеет injection-векторов.
Единственный риск — оскорбительный характер вывода — является ожидаемой
особенностью продукта.
