# Развёртывание matcraft

## Целевая аудитория

Русскоязычные разработчики и энтузиасты, устанавливающие инструмент через
`cargo install`.

## Процесс развёртывания

### Сборка и публикация на crates.io

```sh
cargo publish
```

После публикации пользователи устанавливают:

```sh
cargo install matcraft
```

### Требования для публикации

- Rust toolchain (1.85+)
- Аккаунт на crates.io
- API-токен crates.io (настраивается через `cargo login`)

## Безопасность и секреты

- Секреты отсутствуют. Приложение не использует API-ключи, токены или базы
  данных.
- Единственный аутентификационный токен — crates.io API-токен для публикации.
  Хранится локально в конфигурации Cargo.

## Видимость ошибок и поддержка

- **Баги и проблемы:** GitHub Issues
- **Обратная связь:** GitHub Issues
- **Канал распространения:** crates.io
- **Мониторинг:** отсутствует (хобби-проект, без SLA)

## Бэкапы

В проекте нет пользовательских данных. Бэкапы не требуются. Исходный код
хранится на GitHub и локально у разработчика.
