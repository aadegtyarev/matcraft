# matcraft

CLI-генератор русских матерных выражений по морфологическим правилам.

Генерирует слова на основе корней, приставок, суффиксов и окончаний,
комбинируя их по морфологическим правилам русского языка.

## Установка

```sh
cargo install matcraft
```

## Использование

```sh
matcraft --topic <ТЕМА>
```

Доступные темы: `хуй`, `пизда`, `ебать`, `блядь`.

Пример:

```sh
$ matcraft --topic хуй
захуярить
```

Каждый запуск выбирает случайную комбинацию морфем для указанной темы.

## Разработка

Требуется Rust 1.85+ (edition 2024).

```sh
cargo build
cargo test
```
