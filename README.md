# matcraft

CLI-генератор матерных выражений по теме.

Берёт любую тему (слово или фразу) и генерирует обсценные фразы,
комбинируя тему с матерными корнями по морфологическим правилам
русского языка. Никакого AI, никаких словарей — чистая комбинаторика.

## Установка

```sh
cargo install matcraft
```

## Использование

```sh
# Одна фраза
matcraft --topic птички
# → пиздатые птички

# Несколько фраз
matcraft --topic космос --count 3
# → ебанутый космос
# → космос — пиздец
# → охуеть, космос!

# Любая тема — любое слово или фраза
matcraft --topic философия
# → пиздатая философия
```

### Аргументы

| Аргумент | Описание |
|----------|----------|
| `--topic <ТЕМА>` | Тема генерации — любое слово или фраза (обязательный) |
| `--count <N>` | Количество фраз (по умолчанию 1, макс. 100) |

Поддерживаются многословные темы:
```sh
matcraft --topic "красная площадь"
```

Тема может быть на любом языке, но эвристика рода (для согласования
прилагательных) работает для русских слов.

## Как это работает

1. Вы указываете тему — `matcraft --topic птички`
2. Движок выбирает случайный матерный корень (пизд-, хуй-, еб-, бляд-)
3. Выбирает случайный шаблон фразы:
   - **Прилагательное + тема**: "пиздатые птички"
   - **Междометие + тема**: "охуеть, птички!"
   - **Тема — оценка**: "птички — пиздец"
4. Прилагательное согласуется с темой по роду
